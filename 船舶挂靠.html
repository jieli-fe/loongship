通过船名、呼号、MMSI、IMO 等条件查询船舶的历史靠港记录。 注意：每次最多查询最近一年（365）天的靠港记录。

# 参数:

参数名称 是否必填 参数说明
type 是 查询挂靠的类型：
1.mmsi(船舶mmsi)
2.callsign(船舶呼号)
3.shipname(船舶名称)
4.imo(船舶imo)
value 是 type 字段对应的值
begin 是 开始时间 时间戳以秒为单位。
默认查询10天内的数据，最大查询365天。
end 是 结束时间
return promise
