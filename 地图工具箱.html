# 测距

# 创建

方法 说明
Ls.control.measure(<Obecjt> options)) 调用测距方法

# 示例

var measure = Ls.control.measure({
    lineColor: '#0088fd',
    lineWeight: 2,
    lineDashArray: '6, 6',
    lineOpacity: 1
})
measure.addTo(map);Highlighted
measure.startMeasure();

# 测距参数

名称 默认值 说明
lineColor 'black' 线条颜色
lineWeight 2 线条的宽度
lineDashArray '6, 6' 可以定义虚线线型
lineOpacity 1 填充透明度

提示

目前只支持line的属性自定义, point自定义暂不支持

# 方法

方法名称 返回值 说明
addTo(<Map> map) 将测距添加到指定地图上
start(<Map> map) 开始测距，调用后，屏幕开始测距
stop() 停止测距,并且清空测距图层
