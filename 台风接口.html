# 获取正在进行的台风

示例：

//回调
var callback = function (data){
    //do something 
}
ty.getCurrentTyphoonData(callback);

//Promise
ty.getCurrentTyphoonData().then((data)=>{

});

//demo
ty.getCurrentTyphoonData().then(()=>{
    //获取正在进行台风的数据，并且插入到 map 地图上
