# 地图初始化

方法 说明
Ls.map(<String> elementId, <Object> options) 创建基础地图

# 示例

var map = Ls.map("map", {
	zoom:12
});
1
2
3

# 参数

名称 默认值 说明
zoom 2 地图缩放层级，2-18
center [31.24, 121.15] 地图中心点[lat, lon]
dragable true 是否能够拖动
zoomable true 是否支持缩放
mapType 'map' 地图类型 map地图 ,sat卫星图 , chart海图
key key 使用数据API的key (请联系客服代表获取key)

# 方法

方法名称 返回值 说明
setOptions(<Object> options) 设置参数
setCenter(<Array> center) 设置中心点
getCenter() [lat, lng] 获取中心点
setZoom(<Number> level) 设置地图层级
getZoom() Number 获取地图层级
setMapType(<String> type) 设置地图类型
setKey(<String> key) 设置使用数据API的key (请联系客服代表获取key)
