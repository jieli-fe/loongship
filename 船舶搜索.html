# 关键词搜索

关键词搜索是通过已知的船舶某些参数，来获取船舶 ShipID 的数据接口。

关键词搜索兼容的类型:

  1. MMSI
  2. IMO
  3. 船舶名称
  4. 呼号

# 参数:

参数名称 是否必填 参数说明
value 是 查询的关键词
max 否 最大返回条数， 默认100条
return Promise

# 示例:

Ls.shipSearch(value, 100).then((data) => {
    // console.log(data)
})

# 单船搜索

单船搜索是已知船舶的 ShipID 之后，进行单船船位获取的方式。

参数:

参数名称 是否必填 参数说明
id 必填 搜索的船舶id
return promise

# 示例:

Ls.shipIDSearch(412044071).then((data)=>{
    // console.log(data)
})

# 多船搜索

参数:

参数名称 是否必填 参数说明
value 是 输入查询的mmsi，最多支持500条。支持两种格式
1，'412044071,412044402,412044063,412044072'; 2, [412044071,412044402,412044063,412044072]
return promise

# 示例:

var value = '412044071,412044402,412044063,412044072';
Ls.shipSuperSearch(value).then((data) => {
    // console.log(data)
}

# 区域船舶搜索

搜索特定区域内的船舶数据。

# 参数:

参数名称 是否必填 参数说明
minlat 度 是 最小纬度。
此查询所有经纬度数据都要乘以100万，用于确保查询的精度
minlng 是 最小经度
maxlat 是 最大纬度
maxlng 是 最大经度
hour 否 最近几小时船位
(单位:小时,可支持小数,如半小时就是0.5,默认24小时)

ps： 这个传值特殊，需要对坐标点的 minlat ,minlng , maxlat ,maxlng 排序。 示例:

var params = {29.9555308205,30.0161940050,121.6829632712,121.8546246482,1}
Ls.shipAreaSearch(params[0], params[1], params[2], params[3], params[4]).then((data) => {
    //console.log(data)
}
