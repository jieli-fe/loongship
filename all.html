# 点 - CircleMarker

CircleMarker本质是在地图上绘制圆形覆盖物，可以通过设置笔画宽度、半径、中间的不透明度以达到绘制环型的效果 CircleMarker以像素为单位，当地图放大时，视觉效果保持不变。

# 创建

方法 说明
Ls.circleMarker(latlng,<Object options> options)) 初始化一个点

# 示例

#绘制点 - 使边框至少是半径的2倍或者隐藏stroke,填充颜色
var point = Ls.circleMarker([29, 124], {
	color: "red",
	radius: 3
}).addTo(map);

#绘制环
var ring = Ls.circleMarker([29, 124], {
	color: "red",
	weight: 1,
	radius: 3,
	fill: false
}).addTo(map);

# 船舶挂靠

通过船名、呼号、MMSI、IMO 等条件查询船舶的历史靠港记录。 注意：每次最多查询最近一年（365）天的靠港记录。

# 参数:

参数名称 是否必填 参数说明
type 是 查询挂靠的类型：
1.mmsi(船舶mmsi)
2.callsign(船舶呼号)
3.shipname(船舶名称)
4.imo(船舶imo)
value 是 type 字段对应的值
begin 是 开始时间 时间戳以秒为单位。
默认查询10天内的数据，最大查询365天。
end 是 结束时间
return promise

# 示例:

Ls.getPortOfCall('mmsi', '414210000', '1561106481', '1561970481').then((data) => {
    // console.log(data)
}

# 区域船舶搜索

搜索特定区域内的船舶数据。

# 参数:

参数名称 是否必填 参数说明
minlat 度 是 最小纬度。
此查询所有经纬度数据都要乘以100万，用于确保查询的精度
minlng 是 最小经度
maxlat 是 最大纬度
maxlng 是 最大经度

# 气象 - 扩展

总提供5类⽓气象数据，继承⾃自Leaflet.TileLay; ⽓气象总共提供5天的预测数据，超过当前时间5天将抛出异常; 预测数据每3⼩小时更更新⼀一次，会根据传⼊入的时间点向前获取最近的数据; 每类⽓气象属于⼀一个瓦⽚片，可以通过设置时间属性来显示预测数据; 5类瓦⽚片同时叠加时，⾃自上⽽而下的顺序为:⻛风 > ⽓气压 > 洋流 > 能⻅见度 > 浪
